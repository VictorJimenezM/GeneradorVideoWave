# Audio Visualizer (Vite + React + Tailwind)

Proyecto base para un visualizador de audio.

## Requisitos

- Node 14.x (compatible con tu versión actual) o superior

## Scripts

```bash
npm install
npm run dev
```

Luego abre la URL que te muestre la terminal.

## Estructura

- `src/components/`: componentes UI (incluye `AudioVisualizer`)
- `src/hooks/`: hooks reutilizables (incluye `useAudioAnalyser`)
